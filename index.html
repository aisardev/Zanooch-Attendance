SETUP REQUIRED

Configure your Firebase database

This app needs a free Firebase Realtime Database to store employee data across all devices. It takes about 5 minutes to set up.

  1. Go to console.firebase.google.com and sign in with a Google account.
  2. Click "Add project" → give it a name → disable analytics → click Create.
  3. In the left sidebar click Build → Realtime Database → Create database → Start in test mode → Enable.
  4. Copy your database URL — it looks like: https://your-project-default-rtdb.firebaseio.com
  5. Open this HTML file in a text editor and paste your URL into FIREBASE_URL at the top of the script.
  6. Re-deploy to Vercel — done!
